# Social Media Module

> This module is appended to CLAUDE.md during setup when the user's role is Social Media.

---

## Social Media-Specific Workflows

### Workflow — Content Calendar

Maintain `research/content-calendar.md` with:
- Post date
- Platform (Instagram, LinkedIn, Facebook, X, TikTok)
- Content type (static, carousel, reel, story, article)
- Topic/caption summary
- Status (Drafted, Approved, Scheduled, Published)
- Performance (reach, engagement, saves, shares) — updated post-publish

### Workflow — Engagement Tracker

Maintain `research/engagement.md` with weekly snapshots:
- Follower count by platform
- Engagement rate
- Top performing post of the week
- Comments/DMs requiring response
- Trending topics or hashtags relevant to client

### Workflow — Content Brief Generation

When the user says "content idea for [topic]":
1. Draft a content brief with: platform, format, hook, caption structure, hashtags, CTA, visual direction
2. Save to `research/content-briefs/[topic-slug].md`
3. Log in `deliverables.md`

### Social Media-Specific Files

| File | Purpose |
|---|---|
| `research/content-calendar.md` | Monthly content calendar |
| `research/engagement.md` | Weekly engagement snapshots |
| `research/content-briefs/` | Individual content briefs |
| `research/brand-voice.md` | Platform-specific tone and voice notes |
| `research/competitors-social.md` | Competitor social media analysis |

### Social Media Reporting Additions

When producing meeting MOMs or status updates, always include:
- Posts published since last meeting
- Engagement highlights (top post, notable comments)
- Follower growth
- Content pipeline status (what's coming next)
- Any platform algorithm or trend changes to note
